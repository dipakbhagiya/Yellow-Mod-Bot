# 🛡️ Yellow Mod Bot

A professional Discord moderation bot built using Python and discord.py.

---

# 🚀 Features

✅ Ping Command  
✅ Kick Members  
✅ Ban Members  
✅ Purge Messages  
✅ Mute / Unmute System  
✅ Warning System  
✅ Auto Ban After 3 Warnings  
✅ User Info Command  
✅ Server Info Command  
✅ Logging System  
✅ Custom Bot Status  
✅ Embed Based Design  

---

# 📦 Requirements

- Python 3.8+
- discord.py
- python-dotenv
